# Fancy-Tools

## Description
Projet de scripts Shell permettant d’automatiser un environnement Git avec alias, fonctions personnalisées et script d’installation.

---

## Prérequis
- Linux / macOS
- Git installé
- Bash ou Zsh
- Compte GitHub

---

## Installation

```bash
git clone https://github.com/deveshkb/Fancy-Tools.git
cd Fancy-Tools
bash install.sh
